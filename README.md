# KONTAK GATEWAY API

Api Gateway Untuk Semua Apps Yang Dikelola VMS

## API LIST

    |ROUTE                           |HTTP|DESCRIPTION                   |
    |--------------------------------|----|------------------------------|
    |/gateway/sendmail?to={apps name}|POST|kirim pertanyaan spesifik apps|
    |/gateway/bukutamu?to={apps name}|GET |list pertanyaan spesifik apps |
